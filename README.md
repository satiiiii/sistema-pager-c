# Sistema Pager em C

Sistema simples de gerenciamento de pedidos desenvolvido em linguagem C, com estrutura modular e foco em organização de código.

## 🚀 Funcionalidades

- Criar pedidos vinculados a um pager
- Listar pedidos em preparo
- Listar pedidos prontos
- Marcar pedidos como prontos (simulando acionamento do pager)

## 🧱 Estrutura do Projeto

```
sistema_pager/
├── src/        # Arquivos fonte (.c)
├── include/    # Headers (.h)
├── output/     # Executável gerado
```

## ⚙️ Compilação

```bash
gcc src/*.c -Iinclude -o output/pager.exe
```

## ▶️ Execução

```bash
output\pager.exe
```

## 🧠 Conceitos aplicados

- Programação estruturada em C
- Separação de responsabilidades
- Modularização com arquivos .h e .c
- Manipulação de entrada/saída
